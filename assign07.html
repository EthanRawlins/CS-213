CS 213

An introduction to Web Development


About CS 213



Prerequisite

CS 165: Introduction to Object-Oriented Programming



Course Objective

Learn internet and Web basics. Create and debug client-side web applications. Recognize elements of effective web site design. Understand the importance of Web standards. Understand the relationship between client and server side applications and services. Become effective at learning Web technologies from tutorials, textbooks, and online references. Learn the strengths and weaknesses associated with web applications.



Assignments

Programming assignments are designed to allow students to be creative! There are no test beds or specified output that must be matched exactly. A set of minimum features and requirements are specified for each assignment. Completing the minimum requirements on an assignment will earn the student a "B" grade. To receive an "A" grade, the student must go beyond the minimum requirements. To become proficient, and to better understand the technology the student needs to go beyond the minimum requirements and continue to use previous technologies in subsequent assignments.



Expectations

Be responsible for your own education. Take responsibility to complete assignments and tests as assigned...Students are encouraged to work with your classmates to learn technologies and share ideas, but all submitted work must be original.








The Assignments


Assignment 02


This is the first webpage assignment. The purpose of this assignment is to create your personal home page and an assignments page that can be used to execute most of the assignments you will be doing in this class.



Assignment 03


The purpose of this assignment is to add features to your original home page (your "assign02.html" file).



Assignment 04

Create a business web site that consists of two web pages...The purpose of this assignment is two fold. First, to demonstrate the cascading effects of styles. Second, to demonstrate how external styles sheets can be used to give your web pages a consistent look and feel.



Assignment 05

Create a nice looking html mortgage calculator using javascript. Your web page must use external level cascading style sheets. It must include event handlers for validating appropriate fields. All validation is done on the client. This program does not require server side validation, so the action attribute is still undefined.



Assignment 06

Create an HTML form using javascript which allows users on the internet to purchase several items that you want to sell. Your code must include event handlers for validating appropriate fields. Use DOM 0 event model. All validation is done on the client. This program does not require server side validation, so the action attribute is still undefined.
